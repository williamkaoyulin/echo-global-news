# ECHO Global 國際安全新聞雷達

這是一個具備同源 Python 後端的公開網站 MVP。請使用伺服器或部署完成的 HTTPS 網址；直接以 `file://` 開啟 `index.html` 只能預覽介面，無法搜尋。

## 目前功能

- 以英文關鍵字搜尋外國英語新聞，可使用 `AND`、`OR`、引號及括號。
- 自訂開始與結束日期，單次最長 93 個日曆日。
- 預設及自訂監測主題；主題、設定與重要標記保存在使用者的瀏覽器。
- 搜尋結果只保留媒體名稱與官方網域同時通過允許清單的來源。
- 排除台灣英文媒體、中共官媒與其他已列入阻擋清單的來源。
- 每則結果提供可點選的原文入口與媒體官網。Google News 連結會轉導至媒體原文。
- 新聞聲量圖為「本次搜尋結果的每日篇數」；沒有推估討論度、點閱數或社群互動數。
- 本次熱門議題依新聞標題關鍵詞分類，可點入查看對應新聞。
- 免費翻譯僅翻譯新聞標題，重點文字是明確標示的標題摘要，不是 AI 全文摘要。
- 來源篩選、重要標記、每日簡報、複製及列印／PDF。
- 手機直式排版與鍵盤焦點回饋。

## 資料來源與一致性

主要來源是 Google News RSS，只在主要來源無法連線、且沒有可用的最近快取時，才嘗試 GDELT。相同搜尋在後端快取 15 分鐘；當即時來源短暫失敗時，會優先顯示 24 小時內的最近快取，並在介面標示。本網站不會以示範資料偽裝成即時新聞。

新聞索引會持續更新，因此超過快取期後重新搜尋，篇數可能隨新收錄、修正或移除而改變。後端會在回傳前再做日期範圍檢查、去重與固定排序。

## 本機啟動

在此資料夾執行：

```bash
python3 server.py
```

然後開啟 `http://127.0.0.1:4173`。

## Render 部署

專案已包含 `render.yaml`、`/health` 健康檢查及同源新聞代理。

1. 將本資料夾放在 GitHub repository 根目錄。
2. 在 Render 選擇 **New > Blueprint** 並連接 repository。
3. 等待 `/health` 通過後，分享 Render 提供的 HTTPS 網址。

Blueprint 使用免費執行個體與新加坡區域。免費執行個體閒置後可能休眠，第一次開啟需等待啟動。

## 生產環境注意事項

目前是無帳號的共用 MVP。若要處理非公開資料、增加通知或長期保留新聞，還需加入登入權限、資料庫、資料保留政策、來源授權檢查及 API 使用配額管理。
