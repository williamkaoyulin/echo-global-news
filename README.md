# ECHO Global 國際安全新聞雷達

具備同源 Python 後端的網站 MVP。請勿用 `file://` 直接開啟 `index.html` 搜尋；離線檔案只能預覽介面，無法呼叫新聞 API。

- 僅限英語媒體的自訂關鍵字搜尋（支援 `AND`、`OR`、引號）
- 自訂開始／結束日期（資料源單次上限約三個月）
- 所有搜尋結果皆保留原始新聞超連結
- 每日新聞聲量圖與跨來源討論度指數
- 聚焦原文可追溯的第一階段，不包含 AI 翻譯摘要
- 國軍／台海相關監測主題
- 來源國家、語言、關聯度與標記
- 熱門議題、地區聲量與每日簡報入口
- 資料源流量受限時自動切換示範資料

## 啟動

在此資料夾執行：

```bash
python3 server.py
```

然後開啟 `http://127.0.0.1:4173`。

## 正式版建議架構

前端可升級為 Next.js；後端以排程服務抓取 GDELT、授權新聞 API 與指定 RSS，寫入 PostgreSQL/OpenSearch。AI 層負責語言辨識、繁中摘要、實體抽取、相似報導聚類與關聯度評分。通知可串 Email、LINE 或 Teams。

正式部署前仍需補上登入權限、後端快取、資料保留政策、新聞來源授權檢查及 AI 金鑰管理。

## Render 團隊部署

專案已包含 `render.yaml`、健康檢查及同源新聞代理。部署流程：

1. 將本資料夾放入 GitHub repository（私人 repository 需授權 Render 存取）。
2. 在 Render 選擇 **New > Blueprint** 並連接該 repository。
3. 等待 `/health` 通過後，將 Render 提供的 HTTPS 網址分享給同事。

Blueprint 使用 Render 免費執行個體與新加坡區域。新聞搜尋結果會在後端快取 15 分鐘；本階段不需要任何 AI 金鑰。免費執行個體閒置後可能休眠，第一次開啟需稍候啟動。
